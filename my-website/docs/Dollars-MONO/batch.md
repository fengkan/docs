# 文件批处理

MONO 从 v.260212 起，支持批量处理视频文件。

## 导出帧率

导出文件的帧率将与视频帧率一致。如果视频帧率为非整数（如 29.97fps），最终结果可能存在微小偏差。

## 处理时间

由于硬盘读写以及逐帧处理等开销，处理时间会比视频本身更长，一般为视频时长的 3 至 10 倍。具体取决于视频的分辨率、帧率、编码格式以及硬件性能。

## 选项

在批处理时，以下选项将仍然有效，

- 面捕模块
- 导出文件夹
- 导出格式

以下选项将自动调整，

- 增强模式默认开启
- 动捕模式默认为全身跳跃模式
- 动捕 Profile 默认不生效
