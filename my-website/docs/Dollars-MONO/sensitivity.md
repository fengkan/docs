# 动捕灵敏度

您可以通过调节动捕灵敏度选项来获得不同的动捕结果。

如果您是用于输出动画，可以将灵敏度调至灵敏，便于还原动作。

如果您是用于直播，可以将灵敏度调至稳定，在保证动作还原度的基础上提高稳定性。
