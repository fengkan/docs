# 增强模式

您可以在选项窗口中勾选“增强模式”选项，以获得更高质量的动捕效果。

![](../img/2024_05_05_20_47_55-Dollars_MONO.png)

在以下方面，增强模式会提供更好的效果，

- 腰部以及脚部追踪更为稳定
- 整体动作还原更为到位

当使用增强模式时，请注意，

- 增强模式会耗费更多 GPU
- 如果画面中出现多个人物，增强模式下追踪更易收到干扰
- 在横屏模式下，人物的活动范围会收到限制，理想范围可参考下图中的红线部分

![](../img/2024_05_05_20_59_11-Dollars_MONO.png)
