# 录制与播放 BVH

## 录制

在道乐师程序中，按下 R 键，即可将动作录制为 BVH 文件。

再次按下 R 键，可以结束录制。

BVH文件将以当前时间戳为文件名存放于道乐师程序根目录下。

![](../img/FqatAo8HDqyF9Q4MCynMGf9YWDZt.png#center)

## 播放

在道乐师程序中，按下 L 键，您可在弹出的对话框中选择希望播放的 BVH 文件。

:::caution 请注意

道乐师仅支持播放自身录制的 BVH 文件。

播放非道乐师录制的 BVH 文件，可能出现“BVH 骨骼不正确”的提示信息。

:::
