# 使用头戴显示器进行动捕

您也可以使用头戴显示器进行动捕。

请打开选项，勾选“头部使用 HMD”

![](../img/2023_11_02_17_37_52-Dollars_VIVA.png#center)

启动道乐师，可以看到程序中显示出头显。

![](../img/FvAjkltW_cXZtfW1RabtVNg2Sg9Y.png#center)

之后您就可以正常使用道乐师了。

![](../img/FoNH-JQ72B-UZCbkyA_uVM3W_B5s.png#center)

如果您希望头戴 VIVE Tracker 进行动捕，请确保该选项没有被勾选。
