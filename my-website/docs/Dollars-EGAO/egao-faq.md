# Dollars EGAO 常见问题

import TOCInline from '@theme/TOCInline';

<TOCInline toc={toc} />

## 是否支持 FBX 导出{#fbx}

Dollars EGAO 现阶段不支持 FBX 导出。

## 是否支持实时传送至 MAYA 等软件{#streaming}

Dollars EGAO 可将面捕结果实时传送至 Unity，虚幻，Virt-A-Mate，iClone8 等软件，但是暂时不支持 MAYA 等 3D 软件。
