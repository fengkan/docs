# Dollars SOMA 常见问题

import TOCInline from '@theme/TOCInline';

<TOCInline toc={toc} />

## 是否能够自定义姿势？

Dollars SOMA 现阶段不支持自定义姿势。如果您希望增加某些姿势，可以通过我们网站上的联系方式联系我们。

## 是否支持多个用户同时识别？

**身体姿势：** 只支持单个用户的追踪与识别。

**手势识别：** 可以同时监测多个用户的手势，但在响应时，程序会优先处理最清晰、画面占比最大的手势。
